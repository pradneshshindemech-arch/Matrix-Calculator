> A lightning-fast, mathematically robust Inverse Matrix Calculator built with Python and NumPy. 

Stop doing matrix algebra by hand. **MatrixInvert.py** takes any $n \times n$ matrix, checks its determinant, and instantly computes its inverse using optimized LU Decomposition. It is built for speed, accuracy, and ease of use.

---

## ✨ Features

* **🚀 Blazing Fast:** Powered by NumPy's highly optimized C/Fortran LAPACK backend.
* **🛡️ Bulletproof Error Handling:** Automatically detects singular matrices (determinant = 0) and prevents crash-inducing zero-division errors.
* **🧠 Smart Math:** Uses Lower-Upper (LU) Decomposition rather than slow manual algebraic adjugate methods.
* **💻 Plug & Play:** Clean, readable, and ready to drop into any Python project.
